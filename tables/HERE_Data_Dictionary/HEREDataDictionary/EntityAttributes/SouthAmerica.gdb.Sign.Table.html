No. Name Full Name Description Data Type Max Chars Lookup Table
1 OBJECTID Object ID ESRI unique ID. OID 4  
2 SIGN_ID Sign ID Sign ID is the HERE Sign ID corresponding to the Signposts entry. Integer 4  
3 DESTINATION_LINK_ID Destination LinkID Destination Link ID is the unique ID for the outgoing Link of the Sign, for which a Sign Text is phonetically transcribed. Integer 4  
4 ORIGINATING_LINK_ID Originating Link ID Unique ID for the origin (incoming) Link of the sign. The Originating Link ID identifies the source link for the sign. The originating link is the link prior to the manoeuvre decision. A manoeuvre decision is a point where the road splits and the driver may be unable to determine the direction to traverse. The Originating Link ID in conjunction with the Destination Link ID associated with the sign can be used for route guidance timing and map display purposes. Integer 4  
5 EXIT_NUMBER Exit Number Contains the transliterated text for the exit number. String 10  
6 ALT_EXIT_NUMBER Alternate Exit Number Alternate Exit Number provides an alternate Exit Number for the sign. String 10  
7 LANGUAGE_CODE Language Code The language associated with the feature name. The Language Code, in conjunction with Prefix, Feature Name, Street Type, and Suffix, can be used in destination selection, route guidance, and map display. String 3 Sign.LANGUAGE_CODE
8 STRAIGHT_ON_SIGN Straight On Sign Straight-on-Sign identifies signs added solely to provide additional guidance information for the road ahead. Straight-on-Sign can be used to differentiate Straight-on Signs from Exit Signs. String 1 Sign.STRAIGHT_ON_SIGN
