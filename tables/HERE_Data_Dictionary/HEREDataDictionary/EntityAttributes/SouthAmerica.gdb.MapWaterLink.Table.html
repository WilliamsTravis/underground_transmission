12 EXPANDED_INCLUSION Expanded Inclusion Identifies cartographic features or links that meet the expanded inclusion criteria. This attribute makes all of the links or cartographic features meeting the expanded inclusion criteria easily identifiable. It allows for the option to not publish these links/ features if the inclusion causes size issues. Can be used to differentiate links included only for limited use from those published with limited use attribution. For example, when a portion of a hiking trail shares geometry with an unpaved road for use by autos, only those links dedicated for hiking are indicated for limited use only. Note: Features that exceed the standard size requirement do not receive Expanded Inclusion. However, features that are flagged as Expanded Inclusion at a lower Coverage Indicator level retain the Expanded Inclusion flag when the Coverage Indicator gets upgraded to a higher level where the Standard Inclusion size overlaps with that of the Expanded Inclusion. For example, if a lake feature flagged with Expanded Inclusion in Canada has a Coverage Indicator value of N4 and is upgraded to N0, it maintains the Expanded Inclusion flag regardless of meeting standard inclusion size for Coverage Indicator value = N0. If the Expanded Inclusion attribute does not apply to the link or cartographic feature, the attribute is not published. Prior to the Expanded Inclusion initiative, Bridges and Tunnels were only included in the database if they are at least 200 metres in length. Expanded Inclusion allows for Bridges and Tunnels between 5 and 200 metres to be included (if they are significant). These newly included Bridges and Tunnels will have the Expanded Inclusion attribute applied. SmallInteger 2 MapWaterLink.EXPANDED_INCLUSION
13 ISO_COUNTRY_CODE ISO Country Code A three-letter country code as defined by the International Organization for Standardization. The ISO Country Code indicates in which country the feature is located. Different layers describe different features. ISO Country Code attribution is applied to Navigable Links, Administrative Hierarchies, and POIs in various layers. String 3 MapWaterLink.ISO_COUNTRY_CODE
