8 UNIT_NAME Unit Name The Unit Name is the name of the unit (suite etc.) associated to the Micro Point Address. String 150  
9 ARRIVAL_LINK_ID Arrival Link ID Arrival Link is the link with the drive-to Link for the Point Address. The Arrival Link represents the link where one would like to guided to when selecting a specific Point Address. Note: There are a few cases where the Arrival Link is a pedestrian-only link. The Arrival Link is used for route calculation and route guidance, and represents the drive-to Link for the Point Address. Integer 4  
10 ARRIVAL_SIDE Arrival Side Arrival Side indicates on which side of the Arrival Link the Point Address is located. String 1 MicroPointAddress.ARRIVAL_SIDE
11 DISPLAY_LAT Display Latitude The Display Latitude is the latitude of the display position of the POI. Double 8  
12 DISPLAY_LON Display Longitude The Display Longitude is the longitude of the display position of the POI. Double 8  
13 ISO_COUNTRY_CODE ISO Country Code A three-letter country code as defined by the International Organization for Standardization. The ISO Country Code indicates in which country the feature is located. Different layers describe different features. ISO Country Code attribution is applied to Navigable Links, Administrative Hierarchies, and POIs in various layers. String 3 MicroPointAddress.ISO_COUNTRY_CODE
