3 UNIT_OF_MEASURE Unit Of Measure Unit Of Measure identifies the unit of measure mostly used in the country to describe road and traffic conditions. This attribute can be used to determine which measurement system to use as the default in a specific country. String 1 MetaCountry.UNIT_OF_MEASURE
4 CURRENCY_PRECISION Currency Precision Currency Precision identifies the number of digits used after the decimal point or comma for the currency in question. SmallInteger 2  
5 LANGUAGE_CODE Language Code The language associated with the feature name. The Language Code, in conjunction with Prefix, Feature Name, Street Type, and Suffix, can be used in destination selection, route guidance, and map display. String 3 MetaCountry.LANGUAGE_CODE
6 DRIVING_SIDE Driving Side Driving Side identifies the legal driving side in a country. Text Usage Driving Side is critical for route calculation and route guidance. String 1 MetaCountry.DRIVING_SIDE
7 CURRENCY_TYPE Currency Type Currency Type identifies the default currency of a country. The Currency Type attribute specifies the type of currency to use, when displaying currency values. Currency Type should be used in conjunction with Currency Precision. String 3 MetaCountry.CURRENCY_TYPE
8 PHONE_COUNTRY_CODE Phone Country Code Phone Country Code identifies the international dialing code for a country. This attribute can be used when displaying telephone numbers for Points of Interest. String 4  
9 PHONE_PREFIX Phone Prefix Phone Prefix is the prefix of a phone number. The phone number prefix precedes the phone area code for international country dialing. This is sometimes a leading “0” and is considered to be part of the area code when dialing inside a country. The prefix is not used when dialing from outside a country. Phone Prefix precedes the area code in a phone number. It can be combined with other phone number components for display or for automatic dialing purposes. String 2  
10 SPEED_LIMIT_UNIT Speed Limit Unit Speed Limit Unit identifies the unit of measure for speed limits. String 1 MetaCountry.SPEED_LIMIT_UNIT

14 ORDER_8_DESCRIPTION Order 8 Description Order 8 Description is the textual description of the Order 8 level in the Country. The description is in the local language of the country. String 35  
15 BUILTUP_DESCRIPTION Built-up Area Description Built-up Area Description is the textual description of the Built-up level in the Country. The description is in the local language of the country. String 35  
16 HOUSE_NUMBER_FORMAT House Number Format House Number Format identifies if the house number (address) precedes or follows the street name. Countries have varied conventions for the representation of addresses. In some countries the house number is written after the street name, in others it is before: for example “Via Napoleone, 6” in Italy (the house number is after the street name) and “555 E Weddell Dr” in the U.S. (the house number is before the street name). This attribute specifies which convention is used for a specific country. String 1 MetaCountry.HOUSE_NUMBER_FORMAT
17 ISO_COUNTRY_CODE ISO Country Code A three-letter country code as defined by the International Organization for Standardization. The ISO Country Code indicates in which country the feature is located. Different layers describe different features. ISO Country Code attribution is applied to Navigable Links, Administrative Hierarchies, and POIs in various layers. String 3 MetaCountry.ISO_COUNTRY_CODE
18 EBU_COUNTRY_CODE EBU Country Code EBU Country Code is the country code defined by the European Broadcasting Union (EBU). String 1  
19 TOLL_SYSTEM_TYPE Toll System Type Identifies toll system types that do not have a toll structure or automatic controls. Integer 4
