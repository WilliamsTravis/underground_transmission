1 OBJECTID Object ID ESRI unique ID. OID 4  
2 ASSOCIATION_ID Association ID Association ID identifies the POIs involved in the association. Association ID is used to group together POIs belonging to the same association. Integer 4  
3 ASSOCIATION_TYPE Association Type Association Type identifies the type of relationship that exists between POIs involved in a POI Association. Association Type can be used for map display and destination selection to inform a user that a relationship between specific POIs exists. String 2 GuidanceAssociation.ASSOCIATION_TYPE
4 LINK_ID Link ID Unique identifier for the Link to which the Street Address relates. Link ID is used to identify each link in the database. Integer 4  
5 DIRECTION Direction Direction is a general condition modifier used for select single-link conditions that indicates the link direction for which a direction-dependent attribute is applicable. Direction is a sub-attribute to direction dependent condition coding. Direction can be used to retrieve the link direction for which the No Overtaking and Protected Overtaking condition applies. String 1 GuidanceAssociation.DIRECTION
6 CALCULATED_IMPORTANCE Calculated Importance The Calculated Importance attribute is a rating on the scale of 0 to10 of the importance of a Feature for route guidance. The higher the rating, the more relevant the Feature is for route guidance. The value is calculated based on the other Natural Guidance attributes. The Calculated Importance can be used to determine the relevance of a Feature for route guidance. When multiple Features are relevant for a manoeuvre, the Feature with the highest Calculated Importance can be selected. The Calculated Importance is published for use by those customers that do not want to interpret the individual sub-attributes of the Importance Indicator attributes themselves. SmallInteger 2  
7 VISIBILITY Visibility The Visibility attribute defines the visibility of the Feature, from the direction of the associated link. The Visibility attribute defines the visibility of the Feature, from the direction of the associated link and can be used to determine the relevance of a Feature for route guidance. Only used for Natural Guidance. SmallInteger 2 GuidanceAssociation.VISIBILITY
8 SEASONAL_DEPENDENCY Seasonal Dependency The Seasonal Dependency attribute indicates whether the importance of a Feature is (negatively) impacted during a specific time period. The Seasonal Dependency attribute defines the type of restriction, the actual Time Period is mandatory. The Seasonal Dependency attribute can be used to determine the relevance of a Feature for route guidance related to the season of the year, limited to a certain time of year. Only used for Natural Guidance. String 1  
9 RELATIVE_DISTANCE Relative Distance The Relative Distance attribute indicates the perceived distance between the Feature and the guidance location. The Relative Distance attribute can be used to determine the relevance of a Feature for route guidance, and if needed to generate an advice specific to the situation. Only used for Natural Guidance. SmallInteger 2 GuidanceAssociation.RELATIVE_DISTANCE
10 POI_ID POI ID Unique ID for the POI. Integer 4  
11 POLYGON_ID Polygon ID Unique identifier for polygon features. Polygon ID can be used to locate all links making up a given polygon. That information, in conjunction with the Side Inclusion for each link in the Feature Name, can be used to build polygons for map display. Integer 4  
12 AGGR_ID Aggregation ID Aggregation ID is a unique identifier for the Aggregated Feature. Integer 4  
13 FEATURE_EXTENT Feature Extent Provides the distance of visibility for an associated feature from the point where the feature becomes visible to the decision point represented by the Junction Guidance point or the Passing Guidance point. Feature Extent contributes to the overall Calculated Importance score of a Guidance Feature. Integer 4  
14 PERMANENCE Permanence Provides a level representing the probable permanence of a feature. Permanence contributes to the overall Calculated Importance of a Guidance Feature. String 1 GuidanceAssociation.PERMANENCE
