8 DISPLAY_CLASS Display Class Display Class is a classification of features that enables a meaningful display of these specific features.The classification is assigned according to the importance of the associated feature. Display Class information can be used to determine which features to display at different zoom levels. Display Class information can be used to prune out icons/landmarks of lower importance for memory-limited devices. Note: Display Class definitions for hydro-graphic features use different criteria for individual classes than Display Class for Landmarks/POIs. SmallInteger 2 MapAdminArea.DISPLAY_CLASS
9 COVERAGE_INDICATOR Coverage Indicator Coverage Indicator is an attribute to a navigable or non-navigable link that indicates a product level that corresponds to the inclusion of database features. It also indicates verification and completeness for a particular feature in the database. String 2 MapAdminArea.COVERAGE_INDICATOR
10 ADMIN_PLACE_ID Admin Place ID The Admin Place ID is a unique identifier of the lowest administrative level in the hierarchy. Integer 4  
11 CLAIMED_BY Claimed By Claimed By indicates which country claims a Named Place POI in a disputed area. It is only coded for areas that are under dispute. String 3  
12 CONTROLLED_BY Controlled By The Controlled By attribute is used to indicate which country is in control of the area in which the Named Place POI is located. It is only coded in areas that are under dispute. String 3  
13 ISO_COUNTRY_CODE ISO Country Code A three-letter country code as defined by the International Organization for Standardization. The ISO Country Code indicates in which country the feature is located. Different layers describe different features. ISO Country Code attribution is applied to Navigable Links, Administrative Hierarchies, and POIs in various layers. String 3 MapAdminArea.ISO_COUNTRY_CODE
