1 OBJECTID Object ID ESRI unique ID. OID 4  
2 ADMIN_PLACE_ID Admin Place ID The Admin Place ID is a unique identifier of the lowest administrative level in the hierarchy. Integer 4  
3 ZONE_ID Zone ID Identifies the unique identifier for a zone. All zones should be recognised, along with administrative areas, for destination selection. They can also be used for map display and reverse geocoding. For reverse geocoding, Zone Types = PA and KA can be used. Zones in the U.S. can be used for map display. Integer 4  
4 CENSUS_TYPE Census Type The Census Type defines the type of census data for an Administrative Area or Zone (e.g., GNIS Feature ID, Census Class Code, etc.). Census Type can be used to identify the type of census data. String 2 Census.CENSUS_TYPE
5 CENSUS_VALUE Census Value The Census Value defines the actual values defined in the Census Data (e.g., GNIS Feature ID, Census Class Code, etc.). Census Value can be used to identify the value that corresponds to the Census Type. String 10  
6 LANGUAGE_CODE Language Code The language associated with the feature name. The Language Code, in conjunction with Prefix, Feature Name, Street Type, and Suffix, can be used in destination selection, route guidance, and map display. String 3 Census.LANGUAGE_CODE
7 CENSUS_NAME Census Name The Census Name defines the name for Census Areas as defined in the Census Data. Census Name can be used to identify the name of the CBSA (metropolitan or micropolitan), NECTA (metropolitan or micropolitan), Metropolitan Division or NECTA Division. String 120
