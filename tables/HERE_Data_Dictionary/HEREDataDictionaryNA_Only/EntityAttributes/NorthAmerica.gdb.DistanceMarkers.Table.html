1 OBJECTID Object ID ESRI unique ID. OID 4  
2 SHAPE Shape ESRI geometry object. Geometry 0  
3 FEATURE_POINT_ID Feature Point ID Feature Point ID is a unique ID for a Distance Marker. Feature Point ID can be used to retrieve the road name relative to the Distance Marker for destination selection and map display. The Street Name may be retrieved from the Street Address layer. Integer 4  
4 STREET_ADDRESS_ID Street Address ID Unique identifier for a named, addressed navigable link. Integer 4  
5 DISTANCE_VALUE Distance Marker Value Distance Marker Value is the actual distance marker value. Distance Marker Value can be used for destination selection and location referencing. String 10  
6 DIRECTION_ON_SIGN Direction On Sign The official directional identifiers assigned to highways. Direction on Sign should be used in conjunction with Feature Name, Name on Road Sign, and Name Route Type. For route guidance, US-101 South should be used and not just US-101 when appropriate. String 1 DistanceMarkers.DIRECTION_ON_SIGN
7 DIRECTION Direction Direction is a general condition modifier used for select single-link conditions that indicates the link direction for which a direction-dependent attribute is applicable. Direction is a sub-attribute to direction dependent condition coding. Direction can be used to retrieve the link direction for which the No Overtaking and Protected Overtaking condition applies. String 1 DistanceMarkers.DIRECTION
8 UNIT_OF_MEASURE Unit Of Measure Unit Of Measure identifies the unit of measure mostly used in the country to describe road and traffic conditions. This attribute can be used to determine which measurement system to use as the default in a specific country. String 1 DistanceMarkers.UNIT_OF_MEASURE
9 ENHANCED Enhanced Enhanced is an attribute to identify if the Point Feature has been verified or has been automatically generated. The Enhanced attribute can be used to differentiate between Point Addresses that are verified, and Point Addresses that are automatically generated but do not necessarily represent a verified position. String 1 DistanceMarkers.ENHANCED
