1 OBJECTID Object ID ESRI unique ID. OID 4  
2 CONDITION_ID Condition ID Condition ID publishes the HERE Condition ID that is represented by the Turn. Integer 4  
3 WEIGHT_RESTRICTION Weight Restriction Weight Restriction specifies the occurrence of a (total) weight restriction on a link or lane limiting access to vehicles exceeding a specific (total) weight. Weight Restriction can be used to restrict access of vehicles to specific links or lanes based on the vehicle’s weight. Integer 4  
4 HEIGHT_RESTRICTION Height Restriction Height Restriction specifies the occurrence of a height restriction on a link or lane limiting access to vehicles exceeding a specific height. Height Restriction can be used to restrict access of vehicles to specific links or lanes based on the vehicle’s height. Integer 4  
5 LENGTH_RESTRICTION Length Restriction The MARC Language Code applicable to the Zone Name on the left side of the Link. Integer 4  
6 WIDTH_RESTRICTION Width Restriction Width Restriction specifies the occurrence of a width restriction on a link or lane limiting access to vehicles exceeding a specific width. Width Restriction can be used to restrict access of vehicles to specific links or lanes based on the vehicle’s width. Integer 4  
7 WEIGHT_PER_AXLE_RESTRICTION Weight Per Axle Restriction Weight per Axle Restriction specifies the occurrence of a weight per axle restriction on a link or lane limiting access to trucks where the weight per axle is exceeding a specific weight. Weight per Axle Restriction can be used to restrict access of trucks to specific links or lanes based on the truck’s weight per axle. Integer 4  
8 NUMBER_OF_AXLES Number Of Axles Number of Axles is used to model signs specifying the number of axles that are restricted. SmallInteger 2 ConditionTransport.NUMBER_OF_AXLES
9 KPRA_LENGTH KPRA Length KPRA is used to model signs specifying the “Kingpin to Rear Axle” length. These signs are only published in North America. Integer 4  
10 HAZARDOUS_MATERIAL_TYPE Hazardous Material Type Hazardous Material Type is a sub-attribute to the Transport Access Restriction or Transport Restricted Driving Manoeuvre condition and defines the type of hazardous good(s) for which a specific link or lane is closed. Hazardous Material Type can be used to detail the Transport Access Restriction to vehicles with specific goods. SmallInteger 2 ConditionTransport.HAZARDOUS_MATERIAL_TYPE
11 HAZMAT_PERMIT_REQUIRED Hazmat Permit Required HazMat Permit Required applies in areas where the driver is required to have a HAZMAT Permit document to legally drive on the road with hazardous cargo. This requirement is typically not posted. SmallInteger 2 ConditionTransport.HAZMAT_PERMIT_REQUIRED
12 DIRECTION_CLOSURE Direction Closure Direction Closure (38) is an attribute to the Transport Access Restriction condition and defines the direction of the link or lane closed for vehicles with specific legal or physical restrictions. Direction Closure can be used to close links or lanes in the specified direction, for trucks with a specific usage or vehicles with specific loads. Direction Closure is only used to close links or lanes for a specific direction, it is not used to ‘open’ links in a specific direction. String 1 ConditionTransport.DIRECTION_CLOSURE
13 TRAILER_TYPE Trailer Type Trailer Type specifies the occurrence of a restriction on a link or lane related to trucks with a specified number of trailers. Trailer Type can be used to restrict access of vehicles to specific links or lanes based on the number of trailers attached to a truck. SmallInteger 2 ConditionTransport.TRAILER_TYPE
14 PHYSICAL_STRUCTURE_TYPE Physical Structure Type Physical Structure Type is used to specify if a Transport Access Restriction is caused by a Bridge (Overpass) or Tunnel (Underpass) or any other type of structure like cables, pipelines etc. Physical Structure Type can be used to refine the guidance in case of physical restrictions present on a road. SmallInteger 2 ConditionTransport.PHYSICAL_STRUCTURE_TYPE
15 TRANSPORT_SPEED_LIMIT Transport Speed Limit The Transport Speed Limit is a Transport Special Speed Situation condition attribute and indicates the speed limit applicable to transportation vehicles. The Transport Speed Limit can be used to alert the truck driver for truck-specific speed limits applicable to the road. SmallInteger 2  
16 TRANSPORT_SPEED_TYPE Transport Speed Type The Transport Speed Type condition attribute indicates the type of transport speed limit. This includes special speed limits for trucks with hazardous goods, weather dependent transport speed limits etc. The Transport Speed Type can be used to give specific speed limit warning messages based on the type of speed limit. SmallInteger 2 ConditionTransport.TRANSPORT_SPEED_TYPE
17 SPEED_LIMIT_TYPE Speed Limit Type Speed Limit Type indicates if the Transport Speed Limit is Legal or Advisory. SmallInteger 2 ConditionTransport.SPEED_LIMIT_TYPE
18 DIRECTION Direction Direction is a general condition modifier used for select single-link conditions that indicates the link direction for which a direction-dependent attribute is applicable. Direction is a sub-attribute to direction dependent condition coding. Direction can be used to retrieve the link direction for which the No Overtaking and Protected Overtaking condition applies. SmallInteger 2 ConditionTransport.DIRECTION
19 WEIGHT_DEPENDENT Weight Dependent Weight Dependent is a general condition attribute used for select single-link conditions and provides the weight related to specific Transport conditions. Weight Dependent is used as a sub-attribute to specific restrictions, and is not used to model weight restrictions on roads. Weight Dependent can be used to display the weight together with the transport condition for which a weight dependency exists. Integer 4  
20 WEATHER_TYPE Weather Type Weather Type provides the weather type related to the Transport Special Speed Situation condition. This sub-attribute indicates the type of weather condition that is affecting the transport speed limit. Weather Type can be used to display the type of weather condition affecting the truckspecific speed limit or to alert the driver for truck-specific speed limits on the road. SmallInteger 2 ConditionTransport.WEATHER_TYPE
21 PREFERRED_ROUTE_TYPE Preferred Route Type Preferred Route Type is an attribute of the Transport Preferred Route condition and defines the type of preferred route published. Preferred Route Type can be used to refine the preferred route to specific trucks or trucks with specific goods. This allows for tuning the preferred route options to the qualifications of the applicable truck. SmallInteger 2 ConditionTransport.PREFERRED_ROUTE_TYPE
22 TIME_OVERRIDE Time Override The Time Override indicates if the Special Speed Limit is only applicable from Dusk to Dawn or Dawn to Dusk. SmallInteger 2 ConditionTransport.TIME_OVERRIDE
