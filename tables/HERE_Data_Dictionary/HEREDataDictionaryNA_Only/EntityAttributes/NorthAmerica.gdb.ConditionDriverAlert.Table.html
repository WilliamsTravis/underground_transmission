3 TRAFFIC_SIGN_TYPE Traffic Sign Type The Traffic Sign Type identifies the type of warning sign. The Traffic Sign Type can be used to give specific driver alert messaging using the sign type. Traffic Sign Type can also be used to display the Traffic Sign. Warning signs can be used in optimizing route calculation. SmallInteger 2 ConditionDriverAlert.TRAFFIC_SIGN_TYPE
4 TRAFFIC_SIGN_VALUE Traffic Sign Value The Traffic Sign modifier Traffic Sign Value provides values visible on the sign related to specific sign types. Traffic Sign Value can be used to display the traffic signs with the exact restriction or information as provided on the sign. String 100  
5 GEN_WARNING_SIGN_TYPE General Warning Sign Type The Traffic Sign modifier General Warning Sign Type is used to specify the nature or a general warning sign (Traffic Sign Type = 22). General Warning Sign Type can be used to explicitly indicate the type of warning for a General Warning sign. SmallInteger 2 ConditionDriverAlert.GEN_WARNING_SIGN_TYPE
6 TRAFFIC_SIGN_CATEGORY Traffic Sign Category Traffic Sign Category identifies the main sign category to which the sign belongs to. Traffic Sign Category can be used to give specific informative messaging. SmallInteger 2 ConditionDriverAlert.TRAFFIC_SIGN_CATEGORY
7 TRAFFIC_SIGN_SUBCATEGORY Traffic Sign Subcategory The Traffic Sign Subcategory is an attribute that identifies the sub-category of TRAFFIC_SIGN_CATEGORY = 1 - Regulatory Sign. Traffic Sign Subcategory can be used to group Traffic Signs related to Priority. SmallInteger 2  
8 SIGN_DURATION Supplemental Sign Duration The Traffic Sign attribute Supplemental Sign Duration indicates additional sign shield that are displayed below the Traffic Sign indicating the duration of the warning. Note: Existing Supplemental Sign Duration is extended to allow for multiple Supplemental Sign Duration attributes for one Traffic Sign. A Language Code field is added to the ConditionText layer to explicitly indicate the language in which the Supplemental Sign Duration is published. File Geodatabase only publishes Latin-1 supplemental sign text. The supplemental sign duration can be used for map display or to indicate the duration of the warning in the driver alert messaging. Integer 4  
9 SIGN_PREWARNING Supplemental Sign Pre-Warning The Traffic Sign condition attribute Supplemental Sign Pre-Warning indicates additional sign shields that are displayed below the Traffic Sign indicating an upcoming warning or regulation. These signs are only included when no additional Traffic Sign is present at the actual start of the warning or regulation. Note: Existing Supplemental Sign Pre-Warning is extended to allow for multiple Supplemental Sign Pre-Warning modifiers for one Traffic Sign condition. A Language Code field is added to ConditionText layer to explicitly indicate the language in which the Supplemental Sign Duration is published. File Geodatabase only publishes Latin-1 sign text. The supplemental sign pre-warning can be used for map display or to indicate a prewarning for an upcoming warning or regulation in the driver alert messaging. Integer 4  
10 SIGN_VALIDITY_TIME Supplemental Sign Validity Time The Traffic Sign condition attribute Supplemental Sign Validity Time indicates that an additional sign shields exists, displayed below the Traffic Sign, indicating a specific time(s) at which the Traffic Sign is applicable. Example: No overtaking for Trucks from 8:30 till 18:00 hours. Existing Supplemental Sign Pre-Warning is extended to allow for multiple Supplemental Sign Pre-Warning modifiers for one Traffic Sign condition. A Language Code field is added to ConditionText layer to explicitly indicate the language in which the Supplemental Sign Duration is published. File Geodatabase only publishes Latin-1 supplemental sign text. Supplemental sign text for validity time can be used to display the supplemental traffic sign with information as present on the sign. Integer 4

17 DIRECTION Direction Direction is a general condition modifier used for select single-link conditions that indicates the link direction for which a direction-dependent attribute is applicable. Direction is a sub-attribute to direction dependent condition coding. Direction can be used to retrieve the link direction for which the No Overtaking and Protected Overtaking condition applies. SmallInteger 2 ConditionDriverAlert.DIRECTION
18 SIGNAL_SIGN_LOCATION Signal/sign Location Signal/Sign Location identifies the location of the Traffic Signal at the intersection. Signal/Sign Location can be used to display the location of the Traffic Signal at the intersection. SmallInteger 2 ConditionDriverAlert.SIGNAL_SIGN_LOCATION
19 RAILWAY_CROSSING_TYPE Railway Crossing Type Railway Crossing Type indicates if the Railway Crossing is protected with a barrier or unprotected without barriers. The supplemental sign displaying validity time can be used for map display. SmallInteger 2 ConditionDriverAlert.RAILWAY_CROSSING_TYPE
20 WEATHER_TYPE Weather Type Weather Type provides the weather type related to the Transport Special Speed Situation condition. This sub-attribute indicates the type of weather condition that is affecting the transport speed limit. Weather Type can be used to display the type of weather condition affecting the truckspecific speed limit or to alert the driver for truck-specific speed limits on the road. SmallInteger 2 ConditionDriverAlert.WEATHER_TYPE
