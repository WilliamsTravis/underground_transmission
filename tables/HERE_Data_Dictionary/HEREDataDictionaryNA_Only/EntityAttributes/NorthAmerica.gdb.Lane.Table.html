No. Name Full Name Description Data Type Max Chars Lookup Table
1 OBJECTID Object ID ESRI unique ID. OID 4  
2 LANE_ID Lane ID The Lane ID is a unique ID for the Lane Object. Integer 4  
3 LINK_ID Link ID Unique identifier for the Link to which the Street Address relates. Link ID is used to identify each link in the database. Integer 4  
4 LANE_NUMBER Lane Number Lane Number identifies the Lane within the Link. Lane attributes can be used to show the driver lane-specific information for the road ahead. SmallInteger 2  
5 LANE_TRAVEL_DIRECTION Lane Travel Direction Lane Travel Direction identifies the legal travel directions for a Lane. Lane Travel Direction can be used in conjunction with the related attributes (listed above) to guide the driver through lanes, alert the driver of upcoming lane changes and show the driver which lane(s) to use. String 1 Lane.LANE_TRAVEL_DIRECTION
6 LANE_TYPE Lane Type Lane Type identifies the type of lane present on the link. Lane Type may be used for route guidance and map display. Integer 4 Lane.LANE_TYPE
7 ACCESS_ID Access ID The Access ID is the Key into the Vehicle Access table, which is used to indicate which Vehicle Types are allowed to use the Link. Integer 4  
8 LANE_DIVIDER_MARKER Lane Divider Marker Lane Modifier Type Lane Divider Marker describes the appearance and type of the lane separators between lanes of a Link. Lane Divider Marker can be used to show the driver the lane separators matching the road ahead. Lane Divider Marker can also be use to provide informational guidance for HOV lanes. The driver can be informed when they are allowed to enter or exit the HOV lane based on the type of Lane Divider Marker. SmallInteger 2 Lane.LANE_DIVIDER_MARKER
9 CENTER_DIVIDER_MARKER Center Divider Marker Lane Modifier Type Center Divider Marker describes the type of separator for centre dividers on bi-directional Links. Center Divider Marker can be used to show the driver the type of centre divider matching the road ahead. SmallInteger 2 Lane.CENTER_DIVIDER_MARKER
10 DIRECTION_CATEGORY Direction Category Lane Modifier Type Direction Category refers to the arrow direction shown on a sign or by road markings for a specific lane. Direction Category is a bitmask, since multiple types of arrows can exist on one Lane. Direction Category can be used in map display to show how the lanes are marked in reality. When combining the Direction Category values for each specific lane, an application can generate arrow direction pictures. Integer 4  
11 TRANSITION_AREA Transition Area Transition Area defines the stretch of road where the number of lanes changes and lane markings are not present on the road. Transition Area can be used for display purposes to inform the user that the number of lanes changes and show when the transition starts and ends. String 1 Lane.TRANSITION_AREA
12 LANE_FORMING_ENDING Lane Forming/ending Lane Forming/Ending specifies lane forming and lane ending when the number of lanes changes at transition points and transition areas (between two consecutive links). Lane Forming/Ending can be used for display to determine on which side of the road a lane if forming. SmallInteger 2 Lane.LANE_FORMING_ENDING
13 FROM_SPEED_LIMIT From Speed Limit From/To Speed Limit is the posted or legal speed limit in each travel direction on a road. Speed Limit may be used to estimate link traversal times, to prioritise link selection during route calculation, to calculate timing of the route guidance, and for display purposes. SmallInteger 2  
14 TO_SPEED_LIMIT To Speed Limit From/To Speed Limit is the posted or legal speed limit in each travel direction on a road. Speed Limit may be used to estimate link traversal times, to prioritise link selection during route calculation, to calculate timing of the route guidance, and for display purposes. SmallInteger 2  
15 HEIGHT_RESTRICTION Height Restriction Height Restriction specifies the occurrence of a height restriction on a link or lane limiting access to vehicles exceeding a specific height. Height Restriction can be used to restrict access of vehicles to specific links or lanes based on the vehicle’s height. Integer 4  
16 WIDTH Lane Width Restriction Lane Width Restriction specifies the width of the lane when it differs from country specific values. Lane Width Restriction can be used to specify the lane width when it differs from the country specific lane width. Integer 4  
17 LANE_CROSSING_RESTRICTION Lane Crossing Restriction Lane Crossing Restriction is an attribute indicating on which side(s) of a lane it is not allowed to cross into an adjacent lane. When Lane Crossing Restriction does not publish any value, it is assumed that crossing between lanes is allowed. the functional usage is turn-by-turn lane guidance related to HOV lanes. SmallInteger 2 Lane.LANE_CROSSING_RESTRICTION
