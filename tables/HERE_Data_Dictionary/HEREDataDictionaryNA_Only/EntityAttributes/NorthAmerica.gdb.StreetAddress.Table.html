1 OBJECTID Object ID ESRI unique ID. OID 4  
2 STREET_ADDRESS_ID Street Address ID Unique identifier for a named, addressed navigable link. Integer 4  
3 LINK_ID Link ID Unique identifier for the Link to which the Street Address relates. Link ID is used to identify each link in the database. Integer 4  
4 ISO_COUNTRY_CODE ISO Country Code A three-letter country code as defined by the International Organization for Standardization. The ISO Country Code indicates in which country the feature is located. Different layers describe different features. ISO Country Code attribution is applied to Navigable Links, Administrative Hierarchies, and POIs in various layers. String 3 StreetAddress.ISO_COUNTRY_CODE
5 LANGUAGE_CODE Language Code The language associated with the feature name. The Language Code, in conjunction with Prefix, Feature Name, Street Type, and Suffix, can be used in destination selection, route guidance, and map display. String 3 StreetAddress.LANGUAGE_CODE
6 STREET_NAME Street Name The Street Name attribute is the full spelling of the street name including the Street Name Prefix, Street Name Suffix and Street Type. String 100  
7 BASE_NAME Base Name The Base Name part of the Street Name. String 60  
8 STREET_TYPE Street Type The full street type of the Street Name (e.g., Avenue del la, Via della or Ave). String 30  
9 STREET_TYPE_FULL_SPELLING Street Type Full Spelling Street Type Full Spelling stores the full spelling of the Street Type. String 50  
10 STREET_TYPE_BEFORE Street Type Before Street Type Before indicates if the Street Type is before or after the Base Name. Street Type Before can be used to determine the placement of the Street Type for usage in destination selection, map display and route guidance. String 1  
11 STREET_TYPE_ATTACHED Street Type Attached Street Type Attached indicates whether the Street Type is attached or separated by a space fromthe Base Name. Street Type Attached can be used to determine the location of the Street Type for usage in destination selection, map display and route guidance. String 1  
12 PREFIX Prefix Prefix is a directional identifier that precedes the Feature Name of a road. String 10 StreetAddress.PREFIX
13 SUFFIX Suffix Suffix is a directional identifier that follows, but is not included in, the Feature Name of a road. String 10 StreetAddress.SUFFIX
14 ROUTE_TYPE Route Type The route type indicates that the road's name is actually a route number and in many countries is displayed in a shield symbol (i.e. Interstate and State routes in the U.S.). Name Route Type can be used for display and map publication to display the appropriate route shield when used in conjunction with the Direction on Sign. SmallInteger 2 StreetAddress.ROUTE_TYPE
15 DIRECTION_ON_SIGN Direction On Sign The official directional identifiers assigned to highways. Direction on Sign should be used in conjunction with Feature Name, Name on Road Sign, and Name Route Type. For route guidance, US-101 South should be used and not just US-101 when appropriate. String 10 StreetAddress.DIRECTION_ON_SIGN
16 IS_NAME_ON_ROAD_SIGN Is Name on Road Signe Name on Road Sign indicates the name assigned to a road by the official organisation responsible for the existence and maintenance of the road. Name on Road Sign is generally the name posted on the street signs. For destination selection any name may be used. However only names with Name on Road Sign = Y and Exit Number = Y should be used for route guidance. A link may have more than one name with Name on Road Sign = Y. There is no distinction in this case to which name is more important than the other name. String 1 StreetAddress.IS_NAME_ON_ROAD_SIGN
17 IS_EXPLICATABLE Is Explicatable Explicatable indicates which names should be used to identify a given road when providing directions. Names with Name on Road Sign = Y or Exit Number = Y are always suitable for explication. Other names on a link may or may not be. The Explicatable flag will show this. Names which are Explicatable = N are included in the database to assist in destination selection but are not recommended for route guidance. String 1 StreetAddress.IS_EXPLICATABLE
18 IS_JUNCTION_NAME Is Junction Name Junction Name identifies a ramp system that comprises a named Junction between two or more motorways or a named motorway exit/entrance. Junction Name allows for explication of an exit or entry ramp that comprises a named junction without having to use the sign text. String 1 StreetAddress.IS_JUNCTION_NAME
19 IS_EXIT_NAME Is Exit Name Exit Name identifies a name that is an (alpha)numeric exit name. An exit number (Exit Name = Y) as a feature name can be used for destination selection. String 1 StreetAddress.IS_EXIT_NAME
20 IS_STALE_NAME Is Stale Name Stale Name identifies old names that are still in use. Stale Name records are included in the database to assist in destination selection but should never be used during route guidance or map display. String 1 StreetAddress.IS_STALE_NAME
21 IS_POSTAL_NAME Is Postal Name Postal Name identifies additional names that are added to a link merely to match a postal service file. They could not be verified on any other sources or by field research. Postal Names are included in the database to assist in destination selection but should never be used during route guidance or map display. String 1 StreetAddress.IS_POSTAL_NAME
22 IS_VANITY_NAME Is Vanity Name Vanity Name identifies an alternative name (specific to a business, settlement, location, etc.) other than the official street name. The Vanity Name may only apply to one building along the street (e.g. IBM Plaza). Vanity Names are included in the database to assist in address identification and may be used for route guidance. String 1 StreetAddress.IS_VANITY_NAME
23 IS_SCENIC_NAME Is Scenic Name Scenic Route Name identifies a unique name that represents a Scenic Route. Scenic Route Name can be used for destination selection and map display. String 1 StreetAddress.IS_SCENIC_NAME
24 LEFT_START_ADDRESS Left Start Address This is the house number on the left side of the the reference end of the Link. The Left Start Address is used in conjunction with related address attributes and the associated name for destination selection and geocoding. String 10  
25 LEFT_END_ADDRESS Left End Address This is the house number on the left side of the non-reference end of the Link. The Left End Address is used in conjunction with related address attributes and the associated name for destination selection and geocoding. String 10  
26 LEFT_SCHEME Left Scheme The numbering scheme for the addresses assigned to the left side of the link. The Left Address Scheme is used in conjunction with related address attributes and the associated name for destination selection and geocoding. String 1 StreetAddress.LEFT_SCHEME
27 LEFT_ADDRESS_FORMAT Left Address Format The format of the addresses on the left side of the link. The Left Address Format is used in conjunction with related address attributes and the associated name for destination selection and geocoding. String 2 StreetAddress.LEFT_ADDRESS_FORMAT
28 RIGHT_START_ADDRESS Right Start Address This is the house number on the right side of the the reference end of the Link. The right Start Address is used in conjunction with related address attributes and the associated name for destination selection and geocoding. String 10  
29 RIGHT_END_ADDRESS Right End Address This is the house number on the right side of the non-reference end of the Link. The right End Address is used in conjunction with related address attributes and the associated name for destination selection and geocoding. String 10  
30 RIGHT_SCHEME Right Scheme The numbering scheme for the addresses assigned to the right side of the link. The Right Address Scheme is used in conjunction with related address attributes and the associated name for destination selection and geocoding. String 1 StreetAddress.RIGHT_SCHEME
31 RIGHT_ADDRESS_FORMAT Right Address Format The format of the addresses on the right side of the link. The Right Address Format is used in conjunction with related address attributes and the associated name for destination selection and geocoding. String 2 StreetAddress.RIGHT_ADDRESS_FORMAT
32 ADDRESS_TYPE Address Type Address Type identifies the type of the house number range assigned to the link. The Address Type is used in conjunction with related address attributes, and the associated name for destination selection and geocoding. String 1 StreetAddress.ADDRESS_TYPE
33 ROAD_NAME_ID Road Name ID The Road Name ID defines a unique identifier for the Street Name to which the Street Address relates. The Road Name ID can be used to associate Transliterations and Phonetic Data to Street Names. Integer 4  
34 SUBTYPE Subtype ESRI Geodatabase Subtype. SmallInteger 2
