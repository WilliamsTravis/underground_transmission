9 DISPLAY_CLASS Display Class Display Class is a classification of features that enables a meaningful display of these specific features.The classification is assigned according to the importance of the associated feature. Display Class information can be used to determine which features to display at different zoom levels. Display Class information can be used to prune out icons/landmarks of lower importance for memory-limited devices. Note: Display Class definitions for hydro-graphic features use different criteria for individual classes than Display Class for Landmarks/POIs. SmallInteger 2 MapRailroadLink.DISPLAY_CLASS

11 DETAILED_CITY Detailed City This attribute specifies whether a link is part of a Detailed City coverage area. String 1 MapRailroadLink.DETAILED_CITY
