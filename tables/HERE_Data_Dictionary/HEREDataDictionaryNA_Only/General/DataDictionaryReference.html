This page provides descriptions of the data dictionary tables and columns used to describe Layers/Tables, Attributes, and Lookup Tables.

Layer/Table Information

No.NameDescription
1No.A sequential integer value for used for presentation purposes only.
2GeoDatabaseThe ESRI File Geodatabase the layer or table is found in.
3NameThe actual name of the layer (feature class) or table.
4Full NameA more descriptive and meaningful name for the layer or table, typically with abbreviations and acronyms spelled out.
5CategoryHERE data categories including Geocoding Tables, Mapping Layers, Metadata, Point of Interest (POI), Postal Code Centroids and Polygons, Premium Content Tagging Tables, Routing Application Layers and Tables, Routing Tables, Traffic Patterns, Various Content, and Voice Data.
6TypeOne of 4 types: Point, Polyline, Polygon, or Table.
7DescriptionA description of the layer or table, taken from HERE's File Geodatabase Reference Manual, or other HERE documentation.
8DatasetThe ESRI File Geodatabase Dataset the layer or table is found in if applicable.
9RowsThe number of data rows in the layer or table.
10AttributesThe number of attributes in the layer or table.

Attribute Information

No.NameDescription
1No.A sequential integer value for used for presentation purposes only.
2NameThe actual name of the attribute.
3Full NameA more descriptive and meaningful name for the attribute, typically with abbreviations and acronyms spelled out.
4DescriptionA description of the attribute, as taken from the attribute's Definition and Usage sections in HERE's File Geodatabase Reference Manual.
5Data TypeThe data type used for the attribute.
6Max CharsThe maximum number of characters used for text string data types.
7Lookup TableWhen applicable, a hyperlink to a table of code=value pairs, or single values, if the attribute uses a controlled value domain.

Lookup Table Information

No.NameDescription
1No.A sequential integer value for used for presentation purposes only.
2ValueThe actual data values stored in the attribute.
3DescriptionThe meaning of the data values stored in the attribute.
4SpecificationFor a subset of attributes, an additional metadata column of domain value information, from HERE documentation Specification sections.
