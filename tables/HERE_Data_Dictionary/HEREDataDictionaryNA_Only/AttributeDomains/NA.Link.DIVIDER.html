1 1 REF NODE AND LINK 103,530 Divider = 1 is applied when the link and reference node are divided. Left turns (in right-side driving countries, right turns (in left-side driving countries), and U-turns are not allowed from the divided link to/from any link at the reference node or to driveways along the link.

3 L LINK ONLY 141,535 Divider = L is applied when only the link is divided. Turns are allowed at either node of the divided link. U-turns are not allowed along the divided link.
